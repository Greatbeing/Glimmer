# 需求文档 - 微光应用

## 简介

**应用名称**：微光
**创意定位**：每日微光，照亮心房
**应用类型**：移动端优先的每日金句语录推荐与发布平台
**核心功能**：每日推荐、今日发布、个人空间

## 术语表

- **金句**：指富有哲理、优美或激励人心的文字片段，来源于大模型生成或真实文学艺术作品
- **微光捕捉**：用户收藏金句到个人空间的功能
- **BGM**：Background Music，背景音乐

## 功能需求

### 1. 每日推荐

**用户故事**：作为用户，我希望在首页浏览每日推荐的金句语录，配合合适的背景音乐，帮助我在碎片化时间获得心灵滋养。

#### 1.1 金句展示

- 首页以卡片形式展示金句语录
- 金句内容占卡片主体，来源信息（书籍/电影/大模型）显示在底部
- 文字自动匹配背景音乐，音乐控件悬浮在卡片右下角
- 音乐支持播放/暂停切换，显示当前播放状态

**金句来源要求**：
- 大模型动态生成的金句
- 真实文学艺术作品：畅销书籍、电影台词、电视剧对白
- 金句需标注来源类型和具体出处

#### 1.2 滑动交互

- 支持上下滑动切换到下一条金句
- 滑动时平滑过渡动画
- 滑动速度影响切换方向判定

#### 1.3 点赞功能

- 每个金句显示当前点赞数量
- 用户可点击心形图标进行点赞/取消点赞
- 点赞状态实时更新
- 同一用户对同一金句只能点赞一次

#### 1.4 评论功能

- 用户可查看金句的评论列表
- 支持发表简短评论
- 评论显示用户名、评论内容和发布时间
- 评论按时间倒序排列

#### 1.5 捕捉微光

- 提供捕捉微光按钮（星形图标）
- 点击后金句保存到用户个人空间
- 捕捉成功显示动效反馈
- 同一金句不可重复捕捉

### 2. 今日发布

**用户故事**：作为用户，我希望发布每日灵感或心事，记录生活感悟，并获得他人的共鸣。

#### 2.1 发布功能

- 用户输入发布内容（文字上限 500 字）
- 自动附带发布时间戳
- 支持选择心情标签（可选）
- 发布后立即显示在发布列表中

#### 2.2 互动功能

- 其他用户可对发布内容进行点赞
- 其他用户可发表评论
- 显示点赞数和评论数

#### 2.3 内容展示

- 按时间倒序展示所有发布内容
- 显示发布者用户名、发布时间、内容、心情标签
- 支持分页加载更多

### 3. 个人空间

**用户故事**：作为用户，我希望有一个专属空间，保存我捕捉的微光和我自己发布的内容。

#### 3.1 空间入口

- 底部导航栏第三个入口
- 显示用户头像和用户名

#### 3.2 我的捕捉

- 展示用户捕捉的所有金句
- 支持上下滑动浏览
- 可取消捕捉（从个人空间移除）

#### 3.3 我的发布

- 展示用户自己发布的所有内容
- 支持编辑和删除已发布内容
- 显示每条发布的点赞和评论统计

#### 3.4 数据统计

- 显示捕捉总数和发布总数
- 显示收到的总点赞数

## 非功能需求

### 性能要求

- 首页金句卡片切换动画流畅，无明显卡顿
- 图片和音乐资源采用懒加载
- 应用首屏加载时间不超过 3 秒

### 兼容性要求

- 移动端优先设计，支持 iOS 和 Android
- 适配主流全面屏手机
- 支持横屏和竖屏切换

### 数据持久化

- 用户数据存储在本地 localStorage
- 捕捉和发布数据持久化保存
- 支持数据导出功能

## 验收标准

### 每日推荐模块

1. WHEN 用户打开应用，THEN 系统显示每日推荐金句卡片
2. WHEN 用户滑动卡片，THEN 系统平滑切换到下一条金句
3. WHEN 用户点击音乐图标，THEN 系统播放/暂停背景音乐
4. WHEN 用户点击心形图标，THEN 系统切换点赞状态并更新计数
5. WHEN 用户点击星形图标，THEN 系统将金句保存到个人空间并显示捕捉成功动效
6. WHEN 用户点击评论图标，THEN 系统展开评论区并加载评论列表
7. WHEN 用户输入评论并提交，THEN 系统将评论添加到列表并更新评论数

### 今日发布模块

1. WHEN 用户进入今日发布页面，THEN 系统按时间倒序展示发布列表
2. WHEN 用户输入内容并点击发布，THEN 系统将内容保存并显示在列表顶部
3. WHEN 用户点击发布内容的心形图标，THEN 系统切换点赞状态并更新计数
4. WHEN 用户输入评论并提交，THEN 系统将评论添加到该发布内容下

### 个人空间模块

1. WHEN 用户进入个人空间，THEN 系统展示捕捉的金句和发布的內容
2. WHEN 用户在"我的捕捉"中点击已捕捉的金句，THEN 系统从个人空间移除该金句
3. WHEN 用户在"我的发布"中点击编辑，THEN 系统进入编辑模式
4. WHEN 用户在"我的发布"中点击删除，THEN 系统弹窗确认删除后移除该发布

## 技术实现说明

### 技术栈

- 纯前端实现，使用 HTML5 + CSS3 + JavaScript
- 使用 CSS 变量实现主题定制
- 使用 CSS Grid 和 Flexbox 实现响应式布局
- 使用 localStorage 进行数据持久化
- 使用 Web Audio API 播放背景音乐

### 界面设计

- 主色调：暖色调，体现"微光"温暖治愈的理念
- 字体：使用系统默认字体，确保在各平台一致性
- 圆角卡片设计，营造柔和温暖的视觉体验
- 阴影效果增加层次感
